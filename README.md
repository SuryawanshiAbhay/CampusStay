# 🏨 CampasStay – Smart Hostel Booking App

CampasStay is an Android application designed to help students find and book nearby hostels and rooms easily.

## 🚀 Tech Stack
- Java
- Android SDK
- Firebase Authentication (OTP)
- Firebase Realtime Database
- XML UI

## ✨ Features
- User Login
- Owner Add Hostel
- View Available Hostels
- Firebase Integration
- Clean UI Design

## 📌 Architecture
- Activity-based structure
- Firebase backend integration

## 🔮 Future Improvements
- Payment Gateway Integration
- Booking History
- Admin Dashboard
- Ratings & Reviews
